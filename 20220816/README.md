首先什么是预处理命令

#代表着预处理命令，他告诉编译器，在编译源代码之前，要先执行一些操作（编译器会在编译过程开始之前预处理阶段处理这些命令）


编程的风格

1. 首先想好应该用哪些函数来解决问题，定义好空函数。
2. 按照步骤慢慢填充空函数。
